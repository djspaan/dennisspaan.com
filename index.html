Vaardigheden




HTML5 & CSS3

HTML(Hypertext Markup Language) is de “content-markup” taal van het web, waarmee de structuur van de onderdelen aangegeven wordt in een webdocument. CSS(Cascading Style Sheets) geeft aan hoe de onderdelen moeten worden gepresenteerd aan de gebruiker en is de taal die gebruikt wordt om websites stijl te geven.



PHP & MySQL

PHP (PHP: Hypertext Preprocessor) is een scripttaal voor servers en een krachtig hulpmiddel voor het maken dynamische en interactieve webpagina’s. SQL staat voor Structured Query Language, via deze queries is het mogelijk om toegang te krijgen en het bewerken van een database.



Javascript & AJAX

JavaScript is de programmeertaal van HTML en het web. Bied ondersteunende functionaliteit aan een website. AJAX (Asynchronous JavaScript and XML) is een techniek die gebruikt kan worden om bijvoorbeeld een soepele interface zonder de totale pagina te vernieuwen.



Python

Python is een veel gebruikte "high-level" programmeertaal voor allerlei doeleinden. De syntax is zo ontworpen dat de code goed leesbaar is en programmeurs in staat stelt om concepten in minder regels code uit te drukken dan in andere talen mogelijk zou zijn.

Portfolio

dennisbook Social media

Dennisbook is een social media site die ik gemaakt heb om mijn opgedane kennis in praktijk te brengen. De site maakt naast de gebruikelijke HTML, CSS en JavaScript ook gebruik van PHP, AJAX en MySQL. Het logo is ontworpen met de HTML5 canvas techniek. Op dennisbook kun je een profiel aanmaken, personen volgen, gevolgd worden en berichten naar elkaar sturen. Alhoewel de site op het gebied van styling zeker nog wat aandacht kan gebruiken werken de onderliggende technieken wel, en dat is waar het mij om ging. Neem gerust een kijkje!
